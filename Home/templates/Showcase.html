Templates

Go to the User Dashboard and click 'New Template' to create a new template.

Add as many users as you would like to the template. These users and their permissions will be used each time the template is selected.

Edit the Iris Document, selecting where you would like text and each signature to be placed after clicking the buttons.

After saving the template, you can now create a pre-defined Iris Document using this template.

Add the same number of users and specify their name and email in order. Each users signature location will be filled and sent.


Agreements and contracts that your business uses with your clients can be easily implemented as a template. Simply create the template from the User Dashboard menu and add each user, signifying whether they will be allowed to only view the document, sign the document, or edit the document.


When you create a new Iris Document, select the name of the template you would like to use. Fill in each user in the numerical order they were added to the template, adding a name and email to the first user box to signify that all permissions and signatures of the first template user are to be used for this user in the document.


When you create the Iris Document you can also send additional files, set reminders, and schedule the document to be sent at a certain time. All text, brands, and signatures will be added automatically to the document from the preset template. The features available depend on the active subscription on your account.
